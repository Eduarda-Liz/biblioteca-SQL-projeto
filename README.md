# 📚 Sistema de Biblioteca - Projeto SQL

## Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Banco de Dados Relacional utilizando MySQL.
O sistema simula o gerenciamento de uma biblioteca, permitindo o cadastro de autores, editoras, categorias, livros, clientes e empréstimos.
Além da modelagem do banco de dados, foram desenvolvidas consultas para análise das informações armazenadas.

---
## Tecnologias utilizadas
* MySQL
* SQL
---

## Conceitos praticados

* Criação de banco de dados
* Modelagem relacional
* Chaves Primárias (Primary Key)
* Chaves Estrangeiras (Foreign Key)
* Relacionamentos entre tabelas
* Inserção de dados (INSERT)
* Consultas com SELECT
* Filtros utilizando WHERE
* Ordenação com ORDER BY
* Relacionamentos utilizando INNER JOIN
* Agrupamentos com GROUP BY
* Funções de agregação (COUNT e SUM)

---

## Estrutura do Banco de Dados
O banco é composto pelas seguintes tabelas:

### Autor
Armazena informações dos autores dos livros.

### Editora
Armazena informações das editoras.

### Categoria
Responsável pela categorização dos livros.

### Livro
Contém os dados dos livros disponíveis na biblioteca.

### Cliente
Armazena os dados dos clientes cadastrados.

### Empréstimo
Registra os empréstimos realizados pelos clientes.

### Item_Empréstimo
Tabela responsável por relacionar livros aos empréstimos.

---

## Consultas Desenvolvidas

* Livros publicados após o ano 2000.

* Livros e suas categorias.

* Quantidade de livros por categoria.

* Quantidade de empréstimos por cliente.

* Total de livros emprestados por cliente.
